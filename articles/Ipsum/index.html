    Ipsum

    Anyone who has worked doing design work is likely very familiar with the phrase: Lorem Ipsum. For those not in the know however, Lorem Ipsum refers to a set of text that is used as a placeholder. Theis placeholder text allows a designer to quickly add demo text that will be used to help visiualize and modify the flow and layout of a page or document.

    The words Lorem Ipsum themselves are Latin and have the loose translation of "pain itself". However, when designers are using Lorem Ipsum style text, the meaning of the words is not important. If someone were to attempt to translation a full block of Lorem Ipsum text, they would find it to be nonsensical.

    Final Thoughts

    Although at the end of the day, placeholder text is meant to be just a temporary design element rarily seen by the public, I still think and recommend having fun with the process. There are more variations of Ipsum text that can be found online. Consider using one or more of these for you next project.
